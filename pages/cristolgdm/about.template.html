About this website

This website was done "manually". I wanted to try to have a functional website while using as little external packages. As such, while it uses AngularJS and ui-router, it doesn't use any other framework, any package, doesn't have any dependency (edit: I ended up using prism for code previews, as my own implementation was taking way too much of my time). Everything was written by hand.

Same for the CSS: I strongly believe that a website can be good looking AND functional AND fast, without relying on themes or frameworks, and while keeping everything as simple as possible. So again, everything was written by hand. I tried to keep it minimal, but I guess some animations sneaked in.

I hope the experience was comfortable; this is obviously not a big project, but if you're interested you can follow it here. Any advice or feedback is more than welcome!

One more thing: if everything went right, you are currently in a Proxmox VM in a micro-computer in my living room, after being redirected by a whole chain of AWS + pfSense + reverse proxy + dynamic DNS etc. It's not much, but considering how long it took me to make it work I wanted to mention it.
