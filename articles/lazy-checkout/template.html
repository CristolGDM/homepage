If you've been working in a big enough company, you have probably ended up working on branches with beautiful names like RSWPRD-23188-adjust-esp-connection-between-ap-rvc-for-SSU-phase2. Rolls off the tongue.

You could copy paste from git branch every time, or rely on tab completion if your shell support it, but that's still not lazy enough.

Here is what I call my lazy checkout:

checkout(){ BRANCH=$(git branch | grep --max-count=1 $1) BRANCHNOSPACE=`echo $BRANCH | sed 's/ *$//g'` echo "Checking out $BRANCHNOSPACE..." git checkout $BRANCH }

Type any word and it will checkout the first branch to include it. So from now on checkout 23188, checkout esp-conn, checkout phase2 will all be able to directly point you to that branch.

There are probably some improvements to make but I hope it saves you some time :)
